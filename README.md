# Погодный парсер 

Простая программа для получения погоды в любом городе.

---

##Как запустить (очень просто)

### 1. Установи Python
Скачай с [python.org](https://python.org) (версия 3.8+)
**Важно:** при установке поставь галку "Add Python to PATH"

### 2. Скачай проект
Нажми зелёную кнопку **Code** → **Download ZIP**  
Распакуй папку на рабочий стол

### 3. Запусти
**Просто дважды кликни на файл `weather.bat`**

Всё! Программа сама установит библиотеки и запустится.

---

##Как пользоваться

1. Введи название города (например: `Москва`, `Omsk`, `London`)
2. Программа покажет температуру, ветер и влажность
3. Для выхода напиши `/exit`
