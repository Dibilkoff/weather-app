# Погодный парсер 🌤️

Простая программа для получения погоды в любом городе.

## Как пользоваться
1. Установи Python
2. Установи библиотеку: `pip install requests`
3. Запусти: `python weather.py`
4. Введи город, получи погоду
5. Для выхода напиши `/exit`

## Пример:
-----------------------------------
|чтоб выйти напишите '/exit'|
Введите город:лондон
___________________
результаты города 'лондон' показывают:

-----------------------------------
